Scam Shield API

Integrate scam detection into your own products.

Our robust HTTP API allows you to automatically scan URLs in your applications, preventing dangerous links before they reach your users.

curl -X POST https://api.scamshield.com/v1/scan \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -d '{"url":"https://example.com"}'
